# Эксперимент: Сравнение методов моделирования сезонности временных рядов

В работе исследуется, как различные способы кодирования сезонности влияют на качество прогнозирования глобальной табличной модели CatBoost. 
На данных `Electricity Hourly Dataset` сравниваются конфигурации признаков (лаги, сезонные лаги, календарные и фурье-признаки) 
с baseline-моделями (Naive, SeasonalNaive, AutoETS, AutoTheta) на горизонте 168 часов.

**Участники:** Терехин Кирилл Вячеславович, Еремина Елена Дмитриевна

## Структура репозитория

- `results/` - сохраненные результаты (таблицы)
- `src/` - код для обработки данных, запуска экспериментов и подсчета метрик

## Основные выводы

1. **Baseline:** лучший – `SeasonalNaive_168` (MASE168=1.58, sMAPE=17.13). 
Недельная сезонность крайне стабильна, простой сезонный наивный прогноз оказался очень сильным ориентиром.

2. **CatBoost:** явное кодирование сезонности (сезонные лаги, календарные и фурье-признаки) улучшает качество по сравнению с моделью только на обычных лагах. 
Лучшие конфигурации включают календарные лаги + фурье-признаки.

3. **Сравнение с baseline:** ни одна из конфигураций CatBoost не превзошла `SeasonalNaive_168`. 
Причина – высокая стабильность недельного паттерна и совпадение горизонта прогноза (168 ч) с периодом сезонности.

4. **Зависимость от силы сезонности:** выигрыш от сезонных признаков больше на рядах с сильной недельной структурой (группы medium и strong).
